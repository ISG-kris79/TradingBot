# 진입 로직 전체 (v4.1.1)

## 1. PUMP 진입 경로 3개

### 경로 A: PumpScanStrategy (10초 간격)
```
60개 후보 랭킹 (거래량25% + 변동성35% + 모멘텀40%)
  ↓
각 코인 5분봉 150개 분석
  ↓
[차단 조건]
  - 캔들 < 30개 → 차단
  - 가격 < $0.001 → 차단
  - ATR/가격 >= 3% → 차단 (초고변동성)
  - 30분간 +5%+ 상승 → 차단 (이미 늦음)
  - 1시간 저점 대비 +8%+ → 차단 (급등 이후)
  - 마지막 3봉 중 2봉 음봉 → 차단 (하락 시작)
  - 고점 2% 이내 + RSI 65+ → 차단 (꼭대기)
  ↓
[진입 조건] (둘 중 하나)
  1. bullishSignals >= 6 AND RSI 40~75 AND 가격>SMA20
  2. ML >= 65% AND bullish >= 5 AND 모멘텀 AND 고점아님
  ↓
OnTradeSignal → ExecuteAutoOrder("MAJOR_MEME")
```

### 경로 B: SPIKE_FAST (30초 간격 감지)
```
가격 ±3% 변동 감지 (30초 스냅샷)
  + 거래량 $1M+ AND 거래량비율 2x+
  ↓
[차단 조건]
  - 같은 방향 포지션 있음 → 스킵
  - 슬롯 포화 → 차단
  - 블랙리스트 → 차단
  - ATR/가격 >= 3% → 차단
  - 감지 후 +5% 추가 상승 → 차단 (타이밍 만료)
  - RSI >= 75 → 차단 (과열)
  ↓
60초 풀백 대기 (-1% 눌림 시 진입)
  ↓
직접 PlaceOrderAsync (AI Gate 안 거침)
```

### 경로 C: 감시 풀 (거래량 선행 감지)
```
30초 거래량 증분이 평소의 3배+ AND 가격 변동 3% 미만
  ↓
감시 풀 등록 (가격/시간 기록)
  ↓
메인 루프에서 매 사이클 가격 추적
  +0.5~5% 상승 → ExecuteAutoOrder("PUMP_WATCH_CONFIRMED") 마진 100%
  +5% 초과 → 감시 해제 (이미 늦음)
  -1.5% 하락 → 감시 해제
  5분 초과 → 자동 만료
```

## 2. ExecuteAutoOrder 차단 조건 (순서대로)

```
[1] 시그널 유효성 (LONG/SHORT)
[2] 신호 가격 만료 (현재가 대비 1.5%+ 변동 → 취소)
[3] 캔들 데이터 수집
[4] 블랙리스트 체크 (30분, 손절 2회면 당일)
[5] 슬롯 쿨다운
[6] 워밍업 기간
[7] 서킷브레이커
[8] 거래량 < 0.5x 체크
[9] 변동성 ATR/가격 3%+ 차단
[10] 5분봉 고저폭 5%+ 차단
[11] 중복 진입 체크
[12] 메이저 슬롯 4개 포화
[13] PUMP 슬롯 3개 포화
[14] 총 슬롯 7개 포화
[15] AI Gate (ML 65~70% / TF 60~65%)
[16] D1+H4 방향 필터 (메이저만, bias ±1.5)
[17] Survival 모델 (LONG: "손절 먼저" 60%+ → 차단)
[18] 방향 예측기 (LONG: "큰 상승 없음" 60%+ → 차단)
[19] RSI 극단 (88+/12-)
[20] SHORT 5중 필터 (골든크로스/피보/Stoch/SMA60/RSI)
[21] US 세션 16~23시 사이즈 50%
[22] R:R 체크
```

## 6. PUMP 포지션 관리

| 항목 | 값 |
|------|---|
| 마진 | **$200 고정** |
| 레버리지 | 20x |
| 손절 ROE | -40% (가격 -2%) |
| 실가격 손절 | 진입가 대비 -3% |
| 고점 대비 손절 | 최고가 대비 -5% |
| 본절 전환 | ROE +25% |
| 1차 부분익절 | ROE +25% → **40%** 청산 |
| 계단식 1단계 | ROE +25% → 보호선 +15% + 서버 TRAILING_STOP |
| 계단식 2단계 | ROE +80% → 보호선 +40% |
| 트레일링 | ROE +40% 시작, 고점 대비 20% 하락 시 청산 |
| 1분봉 조기 청산 | 3연속 음봉 + RSI<40 + ROE<-10% |
